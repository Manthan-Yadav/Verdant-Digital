# Verdant Consulting - PHP + Bootstrap

This project now runs as a PHP website using:
- PHP
- Bootstrap 5 (CDN)
- Custom CSS
- Vanilla JavaScript

## Project Structure

- `index.php` (root redirect to the website)
- `project/index.php`
- `project/about.php`
- `project/services.php`
- `project/team.php`
- `project/testimonials.php`
- `project/contact.php`
- `project/css/style.css`
- `project/js/script.js`

## Run Locally

Prerequisites: PHP 8+

1. Open terminal in the project root.
2. Start PHP server:

```bash
php -S localhost:8000
```

3. Open in browser:

```text
http://localhost:8000
```

The root `index.php` redirects to `project/index.php`.
